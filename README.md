### Description
A utility application that fetches and calculates real-time foreign exchange rates. This project demonstrates the ability to connect a backend server to external, third-party data sources.

### Tech Stack

Node.js

Express.js

REST APIs

HTML/CSS/JS

### Features

External REST API integration for live exchange rates

Dynamic mathematical calculations based on user input

Responsive user interface for cross-device usage
